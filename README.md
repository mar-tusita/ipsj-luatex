# A template for the LuaTeX class file for the IPSJ Journal

This is an IPSJ Journal class file for LuaTeX based on jlreq, written using expl3. The original `ipsj-luatex.cls` file is available at <https://github.com/attgm/ipsj-luatex>.

No modifications have been made to the original file at this time. Since the original files are licensed under the Apache License, Version 2.0, this file is distributed under the same license. See the LICENSE file for details

For details on how this class file is implemented, see <https://zenn.dev/attgm/articles/8b963d13e6262b>.

This repository provides a template for users who want to use the `ipsj-luatex.cls` class file in other projects.

## Usage

On GitHub, click the "Use this template" button to create a new repository based on this template. You can then use the `ipsj-luatex.cls` file in your project to create documents that conform to the IPSJ Journal style.

You can also refer to the example files provided in the repository for guidance on how to use the class file effectively.

This template has been confirmed to compile with LuaLaTeX (TeX Live 2025) on Windows 11 and on Overleaf as of June 2026.

# IPSJ Journal 向け LuaTeX クラスファイルのテンプレート

これは、jlreq をベースとし、expl3 を用いて記述された LuaTeX 用の IPSJ Journal クラスファイルです。元となる `ipsj-luatex.cls` ファイルは <https://github.com/attgm/ipsj-luatex> にて公開されています。

現時点では、元のファイルには変更を加えていません。元のファイルは Apache License Version 2.0 のもとで配布されているため、本ファイルも同じライセンスで配布されています。詳細は LICENSE ファイルをご確認ください。

このクラスファイルの実装についての詳細は、<https://zenn.dev/attgm/articles/8b963d13e6262b> を参照してください。

このリポジトリは、`ipsj-luatex.cls` クラスファイルを他のプロジェクトで利用したいユーザー向けのテンプレートを提供することを目的としています。

## Usage（使用方法）

GitHub 上で「Use this template」ボタンをクリックすると、このテンプレートをベースに新しいリポジトリを作成できます。その後、このリポジトリ内の `ipsj-luatex.cls` ファイルをプロジェクトで使用することで、IPSJ Journal のスタイルに準拠した文書を作成できます。

また、クラスファイルの使い方については、リポジトリに含まれるサンプルファイルを参照してください。

このテンプレートは、2026年6月時点で、Windows 11 上の LuaLaTeX（TeX Live 2025）および Overleaf でのコンパイル動作を確認しています。
